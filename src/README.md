# React To-Do App ✅

This is a simple To-Do List app built using **React.js**.  
It lets users add and delete tasks — helping them stay productive!

## 🚀 Features
- Add tasks
- Delete tasks
- Clear input field after adding
- Clean and simple UI

## 🛠️ Built With
- React
- HTML5 + CSS3
- JavaScript (ES6+)
- useState & useEffect hooks
